Этот проект содержит автоматизированные тесты для сайта SauceDemo - демо-сайта интернет-магазина для тестирования.

Основные возможности:

✅ Тестирование функциональности авторизации
✅ Проверка различных сценариев логина
✅ Интеграция с Allure для красивых отчетов
✅ Page Object Pattern для удобного поддержания тестов

Предварительные требования
Перед началом работы убедитесь, что у вас установлено:

1. Java Development Kit (JDK)
Версия: JDK 17 или выше
Проверка: java -version
Скачать: Oracle JDK или OpenJDK
2. Apache Maven
Версия: 3.6.0 или выше
3. Git
Версия: 2.20.0 или выше
Проверка: git --version
4. Браузер
Mozilla Firefox - скачать
5. Allure Framework (для отчетов)

Клонировать проект с GitHub
git clone ...
Перейти в папку проекта
cd ...
Сборка проекта
mvn clean compile
mvn dependency:tree

Для запуска тестов нажать кнопку "Run"
